# Notifly 🚀

Notifly is an open-source Android application that forwards notifications from one device to another via Bluetooth. It is the perfect solution for receiving phone notifications on other Android devices or tablets.

[![Notifly Demo](https://img.youtube.com/vi/LHAmIp7WOPs/0.jpg)](https://www.youtube.com/watch?v=LHAmIp7WOPs)

*Click the image above to watch the demo video on YouTube*

## ✨ Key Features
- **Notification Forwarding**: Automatically sends notifications from selected apps to the connected device.
- **Stable Bluetooth Connection**: Uses traditional Bluetooth SPP, ensuring excellent compatibility with many device generations.
- **App Management**: Allows you to filter which applications are permitted to send notifications.
- **Wide Support**: Works from Android 6.0 (API 23) up to Android 16 (API 36).
- **Battery Optimized**: Optimized background connection to minimize battery drain.

## 🛠 Setup & Usage
1. Install the app on both Android devices.
2. Grant **Notification Access** and **Bluetooth** permissions.
3. On the **Sender** device, tap **Select Allowed Apps** to choose which notifications you want to forward.
4. Set one device as the **Sender** and the other as the **Receiver**.
5. Connect the two devices via Bluetooth and start receiving notifications.
